Fetch IKSM bullet token :)

Pass a session token, you are given a bullet token :)

curl -XPOST https://[thissite]/bullet -d '{ sessionToken: "..." }' { "bulletToken": { "token": "..." }, ... }

see /redoc for details.

Stateless

Statelessly implemented. No storage, cache, or user session.
